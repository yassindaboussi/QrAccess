# QR Code Access Management System - Backend

A robust, production-ready backend for managing QR code-based access control systems. Perfect for stadiums, events, gyms, or any facility requiring subscription-based access control.

## 🚀 Features

- ✅ **Complete Authentication System** - JWT-based auth with refresh tokens
- ✅ **User Management** - CRUD operations for users
- ✅ **Subscription Management** - Track active/expired subscriptions
- ✅ **QR Code Generation** - Unique codes with validation
- ✅ **Scan Logging** - Complete audit trail
- ✅ **Role-Based Access Control** - Super Admin, Admin, Scanner roles
- ✅ **Email Notifications** - Send QR codes via email
- ✅ **Analytics** - Track scans and access patterns
- ✅ **Security** - Rate limiting, sanitization, CORS, helmet
- ✅ **Docker Support** - Easy deployment

## 📋 Prerequisites

- Node.js 18+
- MongoDB 6+
- npm or yarn
